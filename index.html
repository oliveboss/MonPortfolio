Création d'API sécurisée - Django & Swagger

Ce projet consiste en la création d'une API sécurisée avec Django pour gérer les entités de l'application web de gestion des rendez-vous pour un garage automobile. L'API permet la communication avec une base de données et inclut des endpoints pour gérer les clients, les mécaniciens, les rendez-vous, et plus encore.

Cette solution offre une architecture robuste et scalable pour gérer efficacement les données et interagir avec l'application Front-End du garage automobile, en garantissant à la fois la sécurité et la performance.

Le projet démontre mes compétences en Développement Back-End avec Django, la création d'APIs sécurisées, et l'utilisation de Swagger pour la documentation automatique des API.

Gestion de bases de données avec Hibernate (Java)

Ce projet est une application Java utilisant Hibernate pour la gestion de bases de données relationnelles. Il comprend la mise en place d'un validateur personnalisé pour garantir l'intégrité des données et permet d'exécuter des actions CRUD (Créer, Lire, Mettre à jour, Supprimer) sur les données de la base de données.

L'application est développée sous Eclipse et utilise Hibernate ORM pour la gestion des objets Java et leur mappage avec les tables de la base de données. Un validateur personnalisé a été créé pour s'assurer que les données saisies respectent des critères spécifiques avant d'être enregistrées dans la base de données. Cela permet d'améliorer la qualité des données et de réduire les erreurs.

Les actions CRUD sont intégrées à l'application, permettant à l'utilisateur de manipuler les données en temps réel via une interface simple. Ces actions sont exécutées avec l'aide de Hibernate pour interagir efficacement avec la base de données, garantissant ainsi des performances et une scalabilité optimales.

Ce projet démontre ma maîtrise de Hibernate, de la gestion des bases de données relationnelles, ainsi que de la programmation Java et des bonnes pratiques en matière de validation et d'interaction avec les données.

Communication entre Machines Virtuelles avec Python

Ce projet consiste à établir une communication réseau entre deux machines virtuelles sur un environnement Linux, en utilisant leurs adresses IP respectives. Le projet a impliqué l'utilisation de Wireshark pour analyser les paquets réseau échangés entre les machines et s'assurer du bon fonctionnement de la communication.

Le code Python développé dans ce projet a permis d'envoyer et de recevoir des messages entre les machines virtuelles via un réseau local. Les adresses IP des machines ont été configurées de manière à faciliter la communication et à assurer une connectivité optimale.

Le suivi des paquets a été effectué à l'aide de Wireshark, qui a permis de visualiser en temps réel les données échangées, d'analyser les protocoles utilisés et de s'assurer de la stabilité et de la sécurité de la connexion. Ce projet met en avant mes compétences en réseaux, en analyse de paquets et en programmation Python.

Ce projet démontre ma capacité à configurer et à gérer des réseaux virtuels, à analyser des données réseau et à écrire des scripts Python pour automatiser les tâches liées à la communication entre machines virtuelles.
